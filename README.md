# Compact-Hexagonal-Fractal-Patch-Antenna-
Design, simulation, and fabrication of a Compact Hexagonal Fractal Microstrip Patch Antenna operating at 2.4 GHz. Developed using ANSYS HFSS with circular fractal slots for antenna miniaturization, improved impedance matching, low return loss, and stable radiation performance for Wi-Fi, Bluetooth, and IoT applications.
